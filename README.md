# Installation 
`Clone the repo`
<br>
Execute the bambi.sh file<br>
```
$ ./Bambi.sh
```
Give the permission to run using <br>
```
$ chmod +x bambi.sh
```
Don't worry nothing malicious, you can check the code. :)<br>
